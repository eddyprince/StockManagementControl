# Quick Start Guide - Stock Management Control System

## 🚀 Getting Started in 5 Minutes

### Step 1: Open in Visual Studio Code

1. **Open VS Code**
2. **File → Open Workspace from File...**
3. Select `StockManagementControl.code-workspace`
4. **Install Recommended Extensions** (VS Code will prompt you)

### Step 2: Restore and Build

```powershell
# Restore NuGet packages
dotnet restore

# Build the project
dotnet build

# Run the project
dotnet run
```

### Step 3: Test the API

1. Open browser: `https://localhost:5001/swagger`
2. Test the WeatherForecast endpoint
3. Verify you get a response

### Step 4: Set Up GitLab (Optional)

Follow instructions in `GITLAB_SETUP.md`
